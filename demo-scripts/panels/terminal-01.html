🔍 Cơ chế Shebang:
   Dòng đầu file:  #!/usr/bin/env node
   → OS đọc #! và gọi: /usr/bin/env node <file>
   → env tìm "node" trong biến $PATH → thực thi
   → Trên Windows: npm tạo file .cmd thay thế
