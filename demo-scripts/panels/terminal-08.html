🔍 path.sep — Ký tự phân cách thư mục (OS-aware):
   Linux/macOS : path.posix.sep   = "/"
   Windows     : path.win32.sep   = "\"
   Hiện tại    : path.sep         = "\"  ← Tự động theo OS

🔍 path.delimiter — Ký tự phân cách biến PATH:
   Linux/macOS : path.posix.delimiter = ":"
   Windows     : path.win32.delimiter = ";"
   Hiện tại    : path.delimiter       = ";"
