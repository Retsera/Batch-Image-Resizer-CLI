 Node.js Core (System/Tooling - KHÔNG LÀM WEB API)
Đặc thù: Sử dụng các module gốc của Node.js để xử lý tác vụ hệ thống.
YÊU CẦU BẮT BUỘC:
Streams & Buffers: Xử lý file lớn (vài GB) mà không bị tràn RAM.
Worker Threads / Cluster: Xử lý đa luồng (ví dụ: nén ảnh, convert video).
File System (fs): Tương tác file mạnh mẽ

Đồ án : Tool CLI để resize ảnh hàng loạt.
